# 用事例Bot
用字例を自動で応答しますっての！(ㆁᴗㆁ✿)

## 概説
4つのモード(追加/更新/削除/検索)があり、<br>
メンション(@ツイート)にどの言葉が含まれているかで処理が変わります。<br>
複数処理の同時実行には対応していません。

- 追加(更新)の必要キーワード:<br>
  1:"見出し(ひらがな)"<br>
  2:"用字例表記"<br>
  3:"解説"<br><br>
  例:<br>
  1:"いま",2:"今",3:"漢字。"を追加(更新)

- 削除の必要キーワード:<br>
  "見出し(ひらがな)"<br><br>
  例:<br>
  "いま"を削除

- 検索の必要キーワード:<br>
  "見出し(ひらがな)"<br>
  ※検索ではモード指定語は不要です<br><br>
  例:<br>
  "いま"
